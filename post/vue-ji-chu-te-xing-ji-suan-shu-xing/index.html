Vue基础特性-计算属性

2021-04-07

通过选择属性computed:{}

里面常常是一个函数
例子

var vm = new Vue({
 el : '#app,
 data: {
　　 firstName : 'Gavin'，
　　 lastName: 'CLY'
 }
 computed : {
　　 fullName : function() {
　　　 // this 指向 vm 实例
　　　 return this.firstName + ' ' + this.lastName
　　 }
　 }
});

计算属性缓存VS方法 比速度

在组件中，计算属性是基于他们的响应式以来进行缓存的，当message不发生变化时，计算属性不会再调用，类似于AJAX的响应式前后端交互
而方法则每次都会调用

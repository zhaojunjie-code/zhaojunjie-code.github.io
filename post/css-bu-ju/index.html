CSS布局

2021-04-17

display:inline-block; 综合了inline和block的神奇效果

inline 使一个元素成为行内元素，行内元素可以设置内外边距，边框厚度；但不能设置宽高，内容只能靠内容撑开；并且不独占一行

block 使一个元素成为块级元素，块级元素可以设置内外边距，边框厚度；可以设置宽高；并且独占一行

inline-block 使一个元素综合行内和块级元素的效果；可以设置内外边距，边框厚度；可以设置宽高；并且不独占一行可以使用

水平对齐

宽度设置的条件下div：magrin:auto;

图片的水平对齐:

img {
    display:block;    //使其成为块级元素
    margin-left:auto;
    margin-right:auto;
}

垂直对齐

最简单的方法：padding加内边距(前提是：这个div没有高度)

另一种方法:line-height等于height，设置行高(前提：单行文本，如果多行，则每一行都有height的行间距，可能会溢出)

！！！ 解决办法:将多行文本用一个标签标起来，
重新设置行高为正常行高;
display:inline-block使其具有行内元素的不占据一行的效果，和block不换行的效果;
最后设置vertical-align:middle;垂直边距为中间！！！

水平垂直对齐

绝对定位+transform(css3)

position:absolute;
left:50%;                                  //水平居中
top:50%;                                 //垂直居中                  //以上对于一个没有宽高的点有用，但是这个元素有宽高，这个居中方法只是是这个元素的左上角居中了
transform:translate(-50%,-50%)                              //使这个元素在自己的宽度基础上x轴向左移动自己的一半，y轴同理
